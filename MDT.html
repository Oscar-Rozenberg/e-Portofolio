Contexte

Chez VGM, nous avons souvent des commandes d'ordinateurs pour des particuliers ou entreprises. Nos fournisseurs nous envoient les ordinateurs sans système d'exploitation. J'ai proposé de monter un serveur MDT / WDS pour automatiser la configuration de ces ordinateurs.

Objectifs

  • Configurer les postes automatiquement après avoir fait le processus de démarrage PXE
  • Possibilité de configurer plusieurs postes en même temps
  • Assurrer une configuration fiable
  • Gains de temps pour la configuration des postes

Suivi de Projet

Nous avons reçu une commande de plusieurs dixaine de PC pour le 16 juillet
